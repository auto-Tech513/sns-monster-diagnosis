# SNS承認欲求モンスター診断 収益化ロードマップ

## 目標

月10万円の広告・関連収益を目標にする。収益は保証できないため、GA4で実測して改善する。

## 必須KPI

- `shindan_start`: 診断開始数
- `shindan_complete`: 診断完了数
- `shindan_share`: X共有クリック数
- `cheki_download`: 結果画像保存数
- 記事ページPV
- 結果画面到達率
- 共有率
- 1000PVあたり収益

## 収益逆算

月間収益 = 月間PV / 1000 * RPM

- RPM 150円の場合: 約667,000PV/月
- RPM 300円の場合: 約334,000PV/月
- RPM 600円の場合: 約167,000PV/月
- RPM 1000円の場合: 約100,000PV/月

診断アプリ単体の広告RPMだけで10万円を安定達成するには、相当な流入が必要。広告に加えて、記事SEO、X共有、アフィリエイト、スポンサー枠、デジタル商品を組み合わせる。

## 実装済み

- 同意後のみGA4/AdSenseを読み込む
- ニックネームや自由入力をGA4へ送らない
- 結果画面に広告枠を設置
- 記事ページに広告枠を設置
- プライバシーポリシー、問い合わせ、運営情報を追加
- 4本の独自コラムを追加
- X共有をポップアップ依存から同一タブ遷移に変更

## 次に必要なもの

- GA4 Measurement ID
- AdSense publisher/client ID
- 結果画面用広告ユニットID
- 記事ページ用広告ユニットID
- 公開後のURL
- AdSense審査通過後の `ads.txt`

## 30日アクション

2. 完了率が70%未満なら質問数、文言、読み込み速度を改善する。
3. 共有率が10%未満ならX文面、結果画像、ボタン位置を改善する。
4. 週3本以上、SNS心理・診断タイプ・SNS疲れ系の記事を追加する。
5. 流入上位記事に記事内広告を1枠だけ追加し、読みやすさを維持する。
6. 広告RPMが低い場合、SNS運用ツール、画像作成ツール、書籍などの関連アフィリエイトを検討する。

### フェーズ1: 即日実装（今週〜2週間）

#### 1. 永久保存機能（既存CUTE2026機構を活用）: ¥120

- 狙い: LINEスタンプと同価格帯にし、心理障壁をほぼゼロにする。
- 体験設計: 60秒時限消滅から「¥120で永久保存」CTAへつなげる。
- 実装方針: Stripe Payment Linksを結果画面に組み込み、購入後に既存の解除機構へ接続する。
- 優先指標: CTA表示数、クリック率、購入率、購入後の保存完了率。

#### 2. 16x16相性診断パック: ¥380

- 狙い: 友人とシェアして盛り上がる追加コンテンツとして、診断直後の熱量を収益化する。
- 提供内容: 自分のタイプの「ベスト相性」と「ワースト相性」を表示する。
- 提供内容: 全16タイプとの相性スコアと簡易コメントを表示する。
- 実装方針: まず静的な相性テーブルで作り、売上が確認できたら解説文を拡張する。
- 優先指標: 購入率、X共有率、相性ページ滞在時間、友人流入数。
