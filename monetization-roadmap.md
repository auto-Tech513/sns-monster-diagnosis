# SNS承認欲求モンスター診断 収益化ロードマップ

## 目標

月10万円の広告・関連収益を目標にする。収益は保証できないため、GA4で実測して改善する。

## 必須KPI

- `shindan_start`: 診断開始数
- `shindan_complete`: 診断完了数
- `shindan_share`: X共有クリック数
- `cheki_download`: 結果画像保存数
- 記事ページPV
- 結果画面到達率
- 共有率
- 1000PVあたり収益

## 収益逆算

月間収益 = 月間PV / 1000 * RPM

- RPM 150円の場合: 約667,000PV/月
- RPM 300円の場合: 約334,000PV/月
- RPM 600円の場合: 約167,000PV/月
- RPM 1000円の場合: 約100,000PV/月

診断アプリ単体の広告RPMだけで10万円を安定達成するには、相当な流入が必要。広告に加えて、記事SEO、X共有、アフィリエイト、スポンサー枠、デジタル商品を組み合わせる。

## 実装済み

- 同意後のみGA4/AdSenseを読み込む
- ニックネームや自由入力をGA4へ送らない
- 結果画面に広告枠を設置
- 記事ページに広告枠を設置
- プライバシーポリシー、問い合わせ、運営情報を追加
- 4本の独自コラムを追加
- X共有をポップアップ依存から同一タブ遷移に変更

## 次に必要なもの

- GA4 Measurement ID
- AdSense publisher/client ID
- 結果画面用広告ユニットID
- 記事ページ用広告ユニットID
- 公開後のURL
- AdSense審査通過後の `ads.txt`

## AdSense以外の即日マネタイズ候補

### 1. もしもアフィリエイト（Amazon/楽天/Yahoo）

- 実装難度: 低。審査通過後は記事リンクや結果後リンクへ自然に差し込める。
- 想定月収: 初期は0円から5,000円、月間1万PV以上で5,000円から30,000円を目標にする。
- 実装手順: もしもアフィリエイトに登録し、Amazon/楽天/Yahoo提携を申請する。
- 実装手順: SNS疲れ、承認欲求、自己分析、スマホ習慣に関係する本やグッズを選ぶ。
- 実装手順: コラム末尾に「診断後に読みたい本」として1枠だけ設置し、広告感を強くしすぎない。

### 2. SUZURIでのキャラクターグッズ販売

- 実装難度: 中。商品作成は簡単だが、売れるデザインと導線の検証が必要。
- 想定月収: 初期は0円から10,000円、Xで拡散した当たりキャラが出れば10,000円から50,000円を目標にする。
- 実装手順: 人気が出やすい診断キャラを3から5体選び、Tシャツ、ステッカー、アクリル系グッズを作る。
- 実装手順: 結果画面下に「このモンスターのグッズを見る」導線を追加する。
- 実装手順: X共有文にグッズ誘導を直接入れず、サイト内の結果後導線から自然に誘導する。

### 3. note有料記事（16タイプ詳細解説）

- 実装難度: 低から中。記事制作の手間はあるが、在庫不要で利益率が高い。
- 想定月収: 初期は0円から20,000円、1記事300円から500円で月200件販売できれば60,000円から100,000円を狙える。
- 実装手順: 16タイプ別に「長所」「SNSで疲れる場面」「付き合い方」「相性」を詳細化する。
- 実装手順: 無料コラムでは概要まで公開し、詳細版をnoteへ誘導する。
- 実装手順: 結果画面にタイプ別noteリンクを置き、診断直後の熱量が高いタイミングで見せる。

### 4. LINE公式アカウント連携

- 実装難度: 中。初期設定は軽いが、継続配信の設計が必要。
- 想定月収: 直接収益は初期0円、リスト獲得後にnote、グッズ、スポンサー誘導で10,000円から50,000円を目標にする。
- 実装手順: LINE公式アカウントを作成し、診断後に「新キャラ追加通知を受け取る」導線を設置する。
- 実装手順: 登録特典として「16タイプ早見表」や「隠し診断キャラ情報」を用意する。
- 実装手順: 週1回までの低頻度配信にし、ユーザー体験を壊さず再訪と有料導線につなげる。
